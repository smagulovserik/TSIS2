# TSIS2 Paint Application

This project is a Paint application built using Pygame.

## Features
- Pencil (free drawing)
- Line tool
- Shapes (rectangle, circle, triangle, etc.)
- Brush size control (1, 2, 3)
- Fill tool (flood fill)
- Text tool
- Save image (Ctrl+S)

## Technologies
- Python
- Pygame
